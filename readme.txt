== Description ==
For daily updates and more about this Plugin keep an eye on oJobs' Facebook Page here: <a href="http://www.facebook.com/oJobsGroup">Like oJobs on Facebook</a></br>
Creative Bengali Contact Form with attachment support and also include Empty Form validation. Before using this plugin, you need to edit this plugin. Please go to "Plugin Editor" and (Bengali Contact Form.php) replace 'yourname@yourdomain.com' with your own email id then save the plugin. Now it's ready to use. You just need to put the shortcode [bcf_form]  in the post or page and it'll be start. Thanks for your support. Good Luck to your success.

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin.
3. Go to your plugin editor and relpace 'yourname@yourdomain.com' this email with your own email ID.
4. Now create a new pages and paste this code [bcf_form]
